# Punto de Venta (POS)

El POS de V8 está diseñado para ser rápido y confiable. Permite a tu personal de servicio tomar pedidos, gestionar mesas y procesar pagos de manera eficiente.

## Flujos de Trabajo Principales

### Toma de Pedidos

- **Servicio Rápido**: Para servicio en mostrador, utiliza el modo "Pedido Rápido" para registrar artículos y cobrar de inmediato.

### Opciones

- **Opciones Obligatorias**: Cuando una receta cuenta con opciones obligatorias, el sistema obliga al personal a realizar selecciones necesarias (ej. "Término de la carne").

## Pagos y Cuentas

### Dividir Cuentas

V8 simplifica la división de cuentas:

- **Dividir entre varias personas**: Divide el monto total entre varias personas, ej. 50-50, dividir entre 3 partes iguales, o diferentes.
- **Dividir entre varios métodos de pago**: Tus clientes pueden utilizar multiples métodos de pago para ordenar, como pagarte una parte en efectivo y otra con tarjeta

### Métodos de Pago

Acepta Efectivo, Tarjetas de Crédito y Pagos Digitales.

## Capacidades Offline

Aunque V8 está construido "Online First", también tiene un modo sin conexión que permite seguir tomando ordenes cuando no tienes internet. El sistema sincroniza todos los datos con la nube una vez que se restaura la conexión.
