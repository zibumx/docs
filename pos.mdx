# Punto de Venta (POS)

El POS de V8 está diseñado para ser rápido y confiable. Permite a tu personal de servicio tomar pedidos, gestionar mesas y procesar pagos de manera eficiente.

## Flujos de Trabajo Principales

### Toma de Pedidos

- **Servicio Rápido**: Para servicio en mostrador, utiliza el modo "Pedido Rápido" para registrar artículos y cobrar de inmediato.
- **Servicio a Mesas**: Asigna pedidos a mesas específicas. Puedes rastrear el número de comensales y el número de asiento (si está configurado).

### Modificadores y Notas

- **Modificadores Obligatorios**: El sistema obliga al personal a realizar selecciones necesarias (ej. "Término de la carne").
- **Solicitudes Especiales**: Agrega notas personalizadas para la cocina sobre alergias o solicitudes específicas.

## Pagos y Cuentas

### Dividir Cuentas

V8 simplifica la división de cuentas:

- **Por Artículo**: Arrastra y suelta artículos en cuentas separadas para cada comensal.
- **Por Asiento**: Si los pedidos se asignaron a asientos, la cuenta se puede dividir automáticamente.
- **División Equitativa**: Divide el monto total en partes iguales entre un grupo.

### Métodos de Pago

Acepta Efectivo, Tarjetas de Crédito y Pagos Digitales. El sistema puede integrarse con terminales específicas para el cálculo automático de montos.

## Capacidades Offline

El POS está construido bajo el principio "Offline First" (Fuera de línea primero).

- **¿Sin Internet? No hay problema**: Puedes seguir tomando pedidos e imprimiendo tickets de cocina incluso si se pierde la conexión a internet. El sistema sincroniza todos los datos con la nube una vez que se restaura la conexión.
